# Atividade no colab - Script Python

Exercicios no colab com foco em usar `for` e `While`

---

## O que foi feito para cada Exercicio realizado:

* **Exercício 1 (Pares e Ímpares):** Usei um laço `for` para passar pelos números de 0 a 100 e uma condição com `% 2` para mostrar na tela quais eram pares e quais eram ímpares.
* **Exercício 2 (Maior e Menor número):** O programa lê três números digitados, guarda eles dentro de uma lista e depois usa as funções `max()` e `min()` do Python para descobrir e mostrar qual é o maior e o menor.
* **Exercício 3 (Nome em escada):** Pega o nome digitado, usa  `for` para ir somando letra por letra, dando o efeito de escada na vertical.
* **Exercício 4 (sequência de Fibonacci):** Fiz um programa com `while` que gera a sequência de Fibonacci até o número de termos que o usuário escolher, calculando sempre o próximo número com base nos dois anteriores.
* **Exercício 5 (Validação de dados):** Criei várias travas com o laço `while`. O programa fica repetindo a pergunta se o usuário digitar algo errado (uma idade menor que zero ou um estado civil que não é existente)
* **Exercício 6 (Número Primo):** Usei o `for` para dividir o número digitado por vários outros números. Se no final ele só puder ser dividido por 1 e por ele mesmo, o programa avisa que ele é primo.
* **Exercício 7 (Fatorial):** Usei o laço `for` configurado de trás para frente para pegar o número digitado e ir multiplicando pelos anteriores até chegar no resultado do fatorial.
* **Exercício 8 (Mexendo com Lista):** Com a lista `L` já definida, fiz códigos com o `for` para descobrir o tamanho dela, somar todos os valores, achar o maior e o menor e, por fim, mostrei a lista organizada em ordem crescente e decrescente.
* **Exercício 9 (Dicionário/Cardápio):** Montei um dicionário no Python para representar um cardapio, separando uma lista para os nomes dos produtos e outra lista para os preços de cada um.
* **Exercício 10 (Validar Senha):** Um sistema simples de login usando `while`. Enquanto o usuário não digitar a senha exata ("676767"), o programa não saira do loop, até estar correto e mostrar o print "Acesso liberado".
* **Exercício 11 (Tabuada):** O programa pede um número de 1 a 10 e, usando o laço `for`, faz as multiplicações de 1 a 10 para exibir a tabuada completa formatada na tela.
